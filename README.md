# Reimbursement Manager

Enterprise-friendly expense and reimbursement management UI for businesses.

Features
- Expense submission with receipt OCR (Tesseract)
- Pending approvals and manager workflows
- Role-based user management and admin controls
- Theme-aware design system with light/dark support

Quickstart
1. Install dependencies:

```bash
npm install
```

2. Run the dev server:

```bash
npm run dev
```

3. Open the app in your browser at: http://localhost:5173

Available scripts
- `npm run dev` — start development server
- `npm run build` — build for production
- `npm start` — serve the production build
- `npm run typecheck` — run type generation and TypeScript check

Project structure
- `app/` — application source (routes, components, styles)
- `public/` — static assets (favicon, images)
- `package.json` — scripts and dependencies
